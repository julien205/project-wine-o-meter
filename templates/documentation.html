Documentation de l'API

You will find here the description of how to use our API

URL: https://wine-o-meter-julien.herokuapp.com/api

This is a very basic API with one Endpoint /api

The input data should be a dictionnary with one key : "input"

the value of this key will be a list of lists(one or more)

you'll have to use a POST method like: res = requests.post(url, json=input)

>>> res = requests.post(https://wine-o-meter-julien.herokuapp.com/api, json={666: [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]})
>>> assert res.status_code == 200
>>> print(res.json())
